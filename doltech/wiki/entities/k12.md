# k12

## English

**One-liner.** English exam-testing platform for high-school & primary students: exam authoring/publishing,
student test execution with REAL/CHILL modes, AI explanations, course integration, and analytics.

**Stack.** Baseline + Apache Tika/POI, Google Sheets, AI queue (`k12-gpt-4o`), CDN, Restate-based AI queue.

**Features.**
- **Exam test management** — `/api/exam-test/*`: CRUD, paged (category/examType/status/search), publish, add-group,
  file validate/import. Entities `K12ExamTest` (`k12_exam_test`) + `k12_exam_test_version`, nested `QuestionGroup[]`/`Question[]`.
- **Student test execution** — `/public/api/do-test/*`: start, save, check-answer (CHILL only), submit, extend-time,
  change-mode (REAL↔CHILL), check-status. Entity `UserExamTestProgress` (`exam_progress`); `AnswerKey[]`.
- **Results & overview** — `/public/api/do-test-overview/*`: result, history, rank.
- **AI explanation (queue)** — `/api/hook/*`: receive result, queue-status, retry-one/all, import. `k12_ai_status`;
  model `k12-gpt-4o` with `read_image_and_answer`; webhook callback.
- **Course integration** — `/api/course-do-test/*` + `/api/learning-management/*`: course-context test + per-student/class stats. `course_exam_progress`.
- **Analytics** — `/api/analyze/progress/{id}`; `Statistic`, `ProgressAnalyzeDTO`.
- **SEO/media + sync/migration** — question SEO, cover images (Tika); `/api/sync/{initial-history,to-event}`, passage migration. `K12ExamTestListener` → [[action-history]].

**Data model.** `k12_exam_test(_version)`, `exam_progress`, `user_exam_test`, `course_exam_progress`, `k12_ai_status`.
Enums: ExamType (OFFICIAL/PRACTICE/…), ExamCategory (HIGH_SCHOOL/PRIMARY_SCHOOL), DoTestMode (REAL/CHILL).

**Integrations.** page-management, document-history, [[action-history]], english-utils, editor-utils, Directus, coursesupporter; Google Sheets, Kafka, Restate AI queue, Redis, S3/CDN.

**Gotchas.** Two test modes (REAL graded vs CHILL answers-visible), switchable mid-test. Score normalized 0-10 (`K12_MAX_SCORE=10`). Two academy vs course flows. Default 4 options/question. Versioned via `DocumentVersionKey`.

## Tiếng Việt

**Một dòng.** Nền tảng test tiếng Anh cho học sinh THPT & tiểu học: soạn/publish đề, làm bài với mode REAL/CHILL,
AI giải thích, tích hợp khoá học, và phân tích.

**Stack.** Baseline + Apache Tika/POI, Google Sheets, AI queue (`k12-gpt-4o`), CDN.

**Chức năng.**
- **Quản lý đề** — `/api/exam-test/*`: CRUD, phân trang, publish, add-group, validate/import file. Entity `K12ExamTest` (+version),
  nested `QuestionGroup[]`/`Question[]`.
- **Học sinh làm bài** — `/public/api/do-test/*`: start, save, check-answer (chỉ CHILL), submit, extend-time, change-mode
  (REAL↔CHILL), check-status. Entity `UserExamTestProgress` (`exam_progress`).
- **Kết quả & overview** — `/public/api/do-test-overview/*`: result, history, rank.
- **AI giải thích (queue)** — `/api/hook/*`: nhận kết quả, queue-status, retry, import. `k12_ai_status`; model `k12-gpt-4o`.
- **Tích hợp khoá học** — `/api/course-do-test/*` + `/api/learning-management/*`: làm bài theo khoá + thống kê học sinh/lớp. `course_exam_progress`.
- **Phân tích** — `/api/analyze/progress/{id}`.
- **SEO/media + sync/migration** — SEO câu hỏi, ảnh bìa; `/api/sync/*`, migration passage. `K12ExamTestListener` → [[action-history]].

**Data model.** `k12_exam_test(_version)`, `exam_progress`, `user_exam_test`, `course_exam_progress`, `k12_ai_status`.

**Tích hợp.** page-management, document-history, [[action-history]], english-utils, editor-utils, Directus, coursesupporter; Google Sheets, Kafka, Restate, Redis, S3/CDN.

**Lưu ý.** Hai mode (REAL chấm điểm vs CHILL hiện đáp án), đổi giữa chừng. Điểm chuẩn hoá 0-10. Hai luồng academy vs khoá học.
Mặc định 4 đáp án/câu. Version qua `DocumentVersionKey`.
