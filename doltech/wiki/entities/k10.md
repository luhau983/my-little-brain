# k10

## English

**One-liner.** Test/exam management & execution platform for English assessments: exam authoring,
Google-Sheets import, student test execution, and AI-powered marking + explanation generation.

**Stack.** Baseline + Google Sheets/Drive, AI queue (GPT-4o), Directus, S3/CDN, page-management.

**Features.**
- **Exam data entry** — `/api/exam-test/*`: CRUD, paged (filters: examTypes, schoolTypes, statuses, used, year),
  publish, clone, mark-used. Entities `ExamTest` (`exam_test`), `ExamTestSection`, `QuestionGroup`, `Question`, `AnswerKey`.
- **Google Sheets import** — `/api/exam-test/file/validate|import` (range A2:AO / A2:AQ, Apache Tika). Returns `ExamTestImportStatisticDTO`.
- **Student test execution** — `/api/public/do-test/*`: start, fetch/save progress, submit (scores + triggers AI marking).
  Entity `UserExamProgress` (`user_exam_progress`): status, userAnswers, timeSpend, statistic, attemptAt.
- **AI marking & explanation** — `/api/hook/*`: queue-status, explanation receive/retry-one/all/import, writing receive.
  Queues `k10-gpt-4o` / `k10-gpt-4o-json`; webhook back; image CDN. Dedup by `examId+questionId`.
- **Dashboard/overview** — `/api/landing/*`: exam-book progress, last-doing, overview by book / by test-ids, filter by status.

**Data model.** `exam_test`, `exam_test_section`, embedded `question_group`/`question`/`answer_option`, `user_exam_progress`, AI-generate tracking.

**Integrations.** page-management, english-utils, editor-utils, s3-service-v2, Directus, Google Sheets/Drive, Kafka/V2, Redis.

**Gotchas.** Public (`/api/public`, students) vs admin (`/api`, JWT). Document versioning via `DocumentVersionKey`.
AI webhooks idempotent by `examId+questionId`. Durations stored as integer seconds. `reOrderCreateIdIfNeeded()` re-indexes questions on save (affects keys).

## Tiếng Việt

**Một dòng.** Nền tảng quản lý & làm bài test tiếng Anh: soạn đề, import Google Sheets, học sinh làm bài, và
AI chấm + sinh giải thích.

**Stack.** Baseline + Google Sheets/Drive, AI queue (GPT-4o), Directus, S3/CDN, page-management.

**Chức năng.**
- **Nhập liệu đề** — `/api/exam-test/*`: CRUD, phân trang (lọc examTypes, schoolTypes, statuses, used, năm), publish,
  clone, mark-used. Entity `ExamTest`, `ExamTestSection`, `QuestionGroup`, `Question`, `AnswerKey`.
- **Import Google Sheets** — `/api/exam-test/file/validate|import` (range A2:AO/A2:AQ, Apache Tika). Trả `ExamTestImportStatisticDTO`.
- **Học sinh làm bài** — `/api/public/do-test/*`: start, lấy/lưu tiến độ, submit (chấm điểm + kích hoạt AI marking).
  Entity `UserExamProgress` (trạng thái, đáp án, thời gian, thống kê, attemptAt).
- **AI chấm & giải thích** — `/api/hook/*`: queue-status, nhận/retry/import giải thích, nhận điểm viết. Queue
  `k10-gpt-4o`(`-json`); webhook trả về; CDN ảnh. Dedup theo `examId+questionId`.
- **Dashboard/tổng quan** — `/api/landing/*`: tiến độ exam-book, đang làm gần đây, tổng quan theo book / theo test-id, lọc theo trạng thái.

**Data model.** `exam_test`, `exam_test_section`, embedded `question_group`/`question`/`answer_option`, `user_exam_progress`.

**Tích hợp.** page-management, english-utils, editor-utils, s3-service-v2, Directus, Google Sheets/Drive, Kafka/V2, Redis.

**Lưu ý.** Public (`/api/public`, học sinh) vs admin (`/api`, JWT). Version qua `DocumentVersionKey`. Webhook AI
idempotent theo `examId+questionId`. Thời lượng lưu bằng số giây. `reOrderCreateIdIfNeeded()` đánh lại index câu hỏi khi save.
